# Resources Directory

This directory contains educational materials, tutorials, and references for the LXAI Mentorship Program 2025.

## Available Resources

- **tutorial.md**: Step-by-step tutorials for key AI/ML concepts
- **references.md**: Curated list of books, papers, and online resources
- **tools.md**: Recommended software tools and platforms
- **career-advice.md**: Career guidance and industry insights

## How to Use These Resources

1. Start with tutorials that match your current skill level
2. Use references for deeper understanding of concepts
3. Explore recommended tools for hands-on practice
4. Review career advice for professional development

## Contributing Resources

Mentors and advanced mentees are encouraged to contribute additional resources that they find valuable. Please submit a pull request with your additions.
